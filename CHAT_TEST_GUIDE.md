# 🧪 WhatsApp 聊天系统 - 测试与使用指南

## 🚀 快速开始

### 1. 启动应用
```bash
cd /workspaces/nomad-echo
streamlit run app.py
```

### 2. 访问应用
```
本地：http://localhost:8501
或显示的网址
```

### 3. 使用流程
```
广场区 → 分享感悟 或 发现感悟 → 选择感悟 → 开启对话 → 对话区
```

## 🐛 故障排查

### 问题：消息全部靠左，没有右对齐

**原因**：HTML 渲染问题或 CSS 未加载
**解决**：
1. 清除浏览器缓存：Ctrl+Shift+Delete
2. 刷新页面：F5
3. 检查浏览器控制台（F12）有无错误
4. 确保 `message-bubble.sent` CSS 存在

### 问题：旅人没有自动回复

**原因**：`simulate_traveler_reply` 函数未执行
**解决**：
1. 查看 Streamlit 控制台有无错误信息
2. 确保 `st.session_state.traveler_typing = True` 被设置
3. 检查延迟是否触发：`time.sleep(1.5)`
4. 确保 `add_message` 函数正常工作

### 问题：打字指示器不显示

**原因**：CSS 动画未定义或 HTML 未生成
**解决**：
1. 检查 `.typing-indicator` 和 `.typing-dot` CSS 是否存在
2. 查看页面源代码（右键 > 查看页面源代码）搜索 `typing-indicator`
3. 确保 `st.session_state.traveler_typing = True` 时显示
4. 检查 `time.sleep(1.5)` 后是否立即清除标志

### 问题：消息丢失或重复

**原因**：数据保存或刷新问题
**解决**：
1. 检查 `chats.json` 文件是否存在和有效
2. 确保 `save_chats()` 函数正确执行
3. 查看 Streamlit 的自动刷新设置
4. 检查磁盘空间是否充足

### 问题：样式看起来不对

**原因**：CSS 加载顺序或浏览器兼容性
**解决**：
1. 使用现代浏览器（Chrome 最新版）
2. 禁用浏览器扩展（如广告拦截器）
3. 尝试无痕/隐私模式
4. 检查 `st.markdown(..., unsafe_allow_html=True)` 是否正确

## 📞 常见问题 (FAQ)

**Q: 为什么我的消息没有立即出现？**
A: 可能网络延迟或 Streamlit 正在重新运行。刷新页面试试。

**Q: 旅人总是说相同的话怎么办？**
A: 这是正常的。点击"重新选择"重新开始对话会有不同的旅人。

**Q: 消息会一直被保存吗？**
A: 是的，所有消息保存到 `chats.json` 文件。刷新页面也能看到历史。

**Q: 能否关闭自动回复？**
A: 可以。编辑 `app.py`，注释掉 `simulate_traveler_reply()` 这行。

**Q: 支持语音消息吗？**
A: 目前不支持，只有文字消息。

**Q: 海报生成需要多久？**
A: 1-2 分钟，取决于 HuggingFace API 的响应速度。
