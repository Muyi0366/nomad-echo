## 🎯 核心功能

### 1. 📝 感悟发射器
**分享你的旅行感悟**

- 📝 自由输入感悟文字（无长度限制）
- 🏷️ 8 种情绪标签选择
  - ✨ 灵感、🌍 冒险、🧘 宁静、💭 思考
  - 🎨 创意、❤️ 感动、🌅 希望、🚀 兴奋
- 💾 自动保存到 `insights_data.json`
- 📊 实时显示已分享条数

### 2. 🔔 即时回响
**发现他人的感悟**

- 🎲 点击按钮随机获取感悟
- 💬 每条感悟下方有"发起对话"按钮
- 🔄 无限回响，每次都是惊喜

### 3. 💬 即时聊天
**与旅人灵魂对话**

- 👥 自动配对虚拟对话伙伴（nomad_XXXX）
- 📨 实时发送和接收消息
- ⏰ 每条消息显示精确时间戳
- 🎨 美观的气泡样式（左右区分）
- 🔄 3 秒自动刷新，即时获取新消息
- 💾 所有对话持久化保存到 `chats.json`

### 4. ✨ 共鸣海报生成器（核心亮点）
**生成灵魂相遇的视觉记忆**

#### 功能描述
- 🤖 AI 智能生成创意海报
- 📝 自动从对话中提取关键词
- 🎬 电影级美学风格
- 📸 高清 800×600 分辨率
- 💬 配文："这是你们灵魂相遇的瞬间"

#### 运行模式

**模式 A：本地美化版（推荐，默认）**
- ✅ 无需 API Key
- ✅ 100% 免费
- ✅ 无网络延迟
- ✅ 即时生成
- 特点：美化的占位海报，固定优雅设计

**模式 B：AI 生成版（可选）**
- 需要 Hugging Face API Key
- 真实 AI 生成，每次不同
- 基于对话内容的个性化设计
- 更高的创意和唯一性

## 🛠️ 配置 API Key（可选）

### 如果想使用 AI 生成版本：

#### 第一步：获取 Token

1. 访问 [Hugging Face](https://huggingface.co)
2. Settings → Access Tokens
3. Create new token → 复制 token

#### 第二步：配置到应用

**方式 1：创建 .env 文件（推荐）**
```bash
cp .env.example .env
# 编辑 .env，填入你的 token
HF_API_KEY=hf_xxxxxxxxxxxxx
```

**方式 2：设置环境变量**
```bash
export HF_API_KEY="hf_xxxxxxxxxxxxx"
streamlit run app.py
```

**方式 3：在应用启动时输入**
- 应用会检测是否有 API Key
- 没有时自动使用本地版本

详见 [POSTER_GUIDE.md](POSTER_GUIDE.md)

## 📊 关键词提取算法

### 规则

1. **分词**：使用正则表达式分割中文字符
   ```python
   pattern = r'[\u4e00-\u9fa5]{2,}'  # 2+ 个连续中文字
   ```

2. **停用词过滤**
   ```python
   stop_words = {"的", "是", "在", "了", "和", "有", ...}
   ```

3. **频率排序**：选择出现频率最高的词

4. **数量限制**：最多 3-5 个关键词

### 示例

| 对话 | 提取结果 |
|------|---------|
| "在尼泊尔山顶看日出，感到生命的意义" | 尼泊尔, 山顶, 日出 |
| "古镇迷路，却找到治愈的茶馆" | 古镇, 迷路, 治愈 |
| "大海、夕阳、我们、灵魂、连接" | 大海, 夕阳, 灵魂 |

## 🐛 故障排除

### 问题 1：海报一直不显示

**检查项**：
- [ ] 是否点击了"生成海报"按钮
- [ ] 是否有聊天消息
- [ ] 浏览器控制台是否有错误

**解决方案**：
```bash
# 清空缓存重启
rm -f .streamlit/cache/
streamlit run app.py
```

### 问题 2：API 返回 503 错误

**原因**：Hugging Face 模型正在加载  
**解决**：等待 1-2 分钟后重试，或使用本地版本

### 问题 3：关键词提取不正确

**原因**：对话内容过短或不含中文  
**解决**：多发几条消息，或确保使用中文
