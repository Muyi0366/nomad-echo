# NomadEcho - 旅行感悟分享平台

## 📖 项目简介

**NomadEcho** 是一个清冷极简的旅行感悟分享平台，让每一个旅行者都能将心中的感悟化作文字，与世界共鸣。

### 核心功能

#### 🚀 感悟发射器
- 输入你的旅行感悟（文字内容）
- 选择对应的情绪标签（8种选择）
- 一键发送，感悟永久保存

#### 🔔 即时回响  
- 点击按钮随机获取他人分享的感悟
- 在他人故事中找到共鸣和灵感
- 实时更新，永远有新的发现

## 🎨 设计特点

- **极简清冷风格**：淡雅渐变背景（灰蓝色系）
- **毛玻璃效果**：现代感十足
- **柔和配色**：主色为紫蓝渐变（#667eea → #764ba2）
- **响应式布局**：两列设计，左发射右回响

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行应用
```bash
streamlit run app.py
```

### 3. 打开浏览器
自动打开 `http://localhost:8501`

## 💾 数据存储

- 所有感悟自动保存在 `insights_data.json`
- 无需配置数据库，开箱即用
- JSON格式便于导出和分析

## 🎯 使用流程

1. **分享感悟**：在左侧输入框输入感悟，选择情绪标签，点击"发送"
2. **获取回响**：在右侧点击"获取他人的感悟"，随机展示一条分享
3. **持续互动**：不断分享和接收，与全球旅人共鸣
