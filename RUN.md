# 🚀 NomadEcho 快速启动指南

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
streamlit run app.py
```

应用会自动打开浏览器，访问 `http://localhost:8501`

---

## ✨ 新增功能详解

### 1️⃣ 聊天数据库（chats.json）

- 所有聊天对话自动保存在 `chats.json` 文件中
- 消息持久化存储，支持多用户对话历史查询
- JSON 格式结构：
  ```json
  {
    "user1_user2": {
      "participants": ["user1", "user2"],
      "messages": [
        {
          "sender": "user1",
          "text": "你好！",
          "timestamp": "2025-01-27 10:30:45"
        }
      ]
    }
  }
  ```

### 2️⃣ 聊天界面

- 在右侧"即时回响"卡片的感悟下方有 **💬 发起对话** 按钮
- 点击后自动生成虚拟对话伙伴（nomad_XXXX）
- 弹出聊天窗口，显示完整的对话历史

### 3️⃣ 自动刷新机制

- 使用 `streamlit-autorefresh` 库实现自动刷新
- 聊天窗口打开时，每 3 秒自动刷新一次以获取新消息
- 支持实时对话体验（模拟接收他人的回复）

### 4️⃣ UI 气泡样式

- **我的消息**（右侧）：紫蓝色渐变气泡
  - 背景：`linear-gradient(120deg, #667eea 0%, #764ba2 100%)`
  - 圆角：`18px 18px 4px 18px`
  - 文字：白色

- **对方消息**（左侧）：白色半透明气泡
  - 背景：`rgba(255, 255, 255, 0.85)`
  - 圆角：`18px 18px 18px 4px`
  - 文字：深灰色 (#2c3e50)
  - 边框：淡蓝色

- **动画效果**：
  - 我的消息从右滑入（slideInRight）
  - 对方消息从左滑入（slideInLeft）
  - 每条消息都有时间戳显示

### 5️⃣ 共鸣海报生成器（新功能！）✨

**功能亮点**：
- 🤖 AI 驱动的创意海报生成
- 📝 自动提取聊天关键词
- 🎨 电影级美学风格
- 📸 本地美化版本（无需 API）
- 🎬 风格："Minimalist film photography, cinematic lighting, healing vibe"

**使用流程**：
1. 与对话伙伴进行聊天
2. 点击 **✨ 生成灵魂回响海报** 按钮
3. 系统自动提取对话关键词
4. AI 生成或本地渲染海报
5. 展示配文："这是你们灵魂相遇的瞬间"

**两种模式**：
- **本地版本**（默认）：美化的占位海报，100% 免费，无网络需求
- **AI 版本**（可选）：调用 Hugging Face API，需配置 API Key

**配置 API Key**（可选）：
```bash
# 方式1：创建 .env 文件
echo 'HF_API_KEY=hf_your_token_here' > .env

# 方式2：设置环境变量
export HF_API_KEY="hf_your_token_here"
```

详细配置见 [POSTER_GUIDE.md](POSTER_GUIDE.md)

## 📱 用户流程

### 第一步：分享感悟
1. 在左侧"感悟发射器"输入文字内容
2. 选择情绪标签（8 种选择）
3. 点击"📤 发送感悟"
4. 感悟保存到 `insights_data.json`

### 第二步：发现他人感悟
1. 在右侧点击"🎲 获取他人的感悟"
2. 随机显示一条其他用户分享的感悟

### 第三步：发起对话（新功能）
1. 在感悟卡片右下方点击"💬"按钮
2. 自动打开聊天窗口
3. 输入消息并点击"📤"发送
4. 消息实时保存到 `chats.json`
5. 点击"❌ 关闭对话"退出聊天

## ⚡ 性能优化建议

当用户数量增多时，可以考虑：

1. **数据库升级**：使用 SQLite 或 MongoDB 替代 JSON
2. **消息分页**：只加载最近 50 条消息
3. **缓存机制**：缓存频繁访问的对话
4. **异步刷新**：使用 WebSocket 替代轮询

## 🐛 故障排除

### 问题：应用启动报错"ModuleNotFoundError"
**解决**：运行 `pip install -r requirements.txt`

### 问题：聊天消息没有实时更新
**解决**：确保浏览器标签页处于活跃状态，Streamlit 会每 3 秒自动刷新

### 问题：消息保存失败
**解决**：检查工作目录权限，确保可以创建/修改 JSON 文件
