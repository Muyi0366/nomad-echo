# 🎨 WhatsApp 聊天界面视觉演示

## 消息气泡对比

### 升级前（使用 st.chat_message）
```
┌─────────────────────────────────────┐
│ 用户消息文本                         │
│ HH:MM:SS                             │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ 旅人消息文本                         │
│ HH:MM:SS                             │
└─────────────────────────────────────┘
```
- 问题：所有气泡都靠左，没有左右区分
- 交互感：低，像消息日志而不是对话

### 升级后（WhatsApp 风格）
```
右对齐，紫色渐变背景：
                    ╭─────────────────╮
                    │ 你的消息        │
                    │ 14:30:45        │
                    ╰─────────────────╯

左对齐，浅蓝背景：
╭─────────────────╮
│ 旅人的回复      │
│ 14:30:50        │
╰─────────────────╯
```
- 优势：清晰的左右区分，就像真实的 WhatsApp
- 交互感：高，自然、直观、现代化

## 气泡样式细节

### 圆角设计
```
一般气泡：         尖角设计：
  ╭──────╮         ╭──────╮
  │      │    →    │      ├─ 右下角 4px (sent)
  ╰──────╯         ╰──────╯

╭──────╮           ╭──────╮
│      │    →    ├─│      │ 左下角 4px (received)
╰──────╯           ╰──────╯
```

### 阴影效果
```
气泡阴影：0 2px 8px rgba(0,0,0,0.08)
- 用户消息：紫色阴影
- 旅人消息：蓝色阴影
- 打字指示器：蓝色阴影

效果：增加立体感，分离对话框
```

## CSS 关键类

### 容器类
- `.chat-container` - 聊天窗口主容器（flex + 滚动）
- `.message-bubble` - 单条消息包装器
- `.message-bubble.sent` - 发送的消息（右对齐）
- `.message-bubble.received` - 接收的消息（左对齐）

### 内容类
- `.bubble-content` - 消息气泡内容盒子
- `.bubble-content.sent` - 发送的内容（紫色）
- `.bubble-content.received` - 接收的内容（蓝色）
- `.bubble-timestamp` - 消息时间戳

### 动画类
- `.typing-indicator` - 打字指示器容器
- `.typing-dot` - 单个跳动的点
- `@keyframes slideIn` - 消息滑入动画
- `@keyframes typing` - 点点跳动动画

```
字体大小：
- 消息：0.95em
- 时间戳：0.75em

间距：
- 消息间距：0.8em
- 容器高度：500px（滚动）

动画：
- 消息出现：slideIn 300ms
- 打字指示器：typing 1400ms
- 旅人延迟：1500ms
```
